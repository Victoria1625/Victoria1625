# Oi, eu sou a Victoria!

Atuo com **Dados, Machine Learning e soluções analíticas aplicadas ao negócio**.

Meu foco está em construir projetos que vão além da modelagem isolada: gosto de transformar dados em **produtos analíticos utilizáveis**, conectando **previsão, backend, visualização e tomada de decisão**.

Hoje concentro meus estudos e projetos em **Ciência de Dados**, com interesse especial em:
- modelos preditivos
- análise de risco
- churn
- people analytics
- soluções de ML aplicadas a problemas reais

## O que eu construo

Tenho trabalhado em projetos de ponta a ponta que envolvem:

- desenvolvimento de modelos de machine learning
- APIs para servir inferência
- interfaces para análise de resultados
- tradução de métricas e outputs técnicos para contextos de negócio
- estruturação de soluções analíticas com foco em usabilidade

Meu objetivo é construir projetos que mostrem não só capacidade técnica, mas também **raciocínio analítico, clareza de decisão e aplicação prática**.

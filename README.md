## Projetos em destaque

### CloudCostOps MLOps

Projeto de **MLOps aplicado a FinOps** para previsão de custos em cloud, detecção de anomalias, alertas de orçamento, data drift, decisão de retreinamento e tracking com MLflow.

Principais pontos:

- previsão de custo diário com XGBoost;
- feature engineering temporal;
- split temporal e backtesting mensal;
- batch forecast;
- monitoramento de performance em produção simulada;
- detecção de anomalias de custo;
- alertas de orçamento;
- detecção de data drift;
- decisão automatizada de retreinamento;
- model metadata;
- MLflow tracking;
- testes automatizados com Pytest;
- CI com GitHub Actions.

O foco principal do projeto é transformar previsões de custo em alertas acionáveis, já que no contexto FinOps o erro mais crítico é subestimar gastos relevantes e perder a oportunidade de agir antes do impacto financeiro.

---

### SmartSLA MLOps

Projeto de **MLOps para previsão de risco de violação de SLA em tickets operacionais**, com foco em priorização preventiva, monitoramento de modelo e decisão de retreinamento.

Principais pontos:

- classificação de risco de violação de SLA com XGBoost;
- ajuste de threshold operacional priorizando recall;
- inferência batch em produção simulada;
- monitoramento de performance;
- detecção de data drift;
- regras de alerta;
- decisão de retreinamento;
- tracking com MLflow;
- testes unitários;
- CI com GitHub Actions.

O foco principal do projeto é reduzir falsos negativos, já que o erro mais crítico no contexto de SLA é deixar passar tickets que provavelmente violariam o prazo.
